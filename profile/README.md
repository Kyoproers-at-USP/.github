# Kyoproers-at-USP
滋賀県立大学競技プログラミングサークルの公式リポジトリです．
## 私たちについて
2024年9月から活動を開始し，同年12月に大学から認可されたサークルです．

"プログラミングに興味がある人達のたまり場"となれるよう，設立しました．

"競技プログラミング"を通して，アルゴリズム力・実装力の向上を目指した活動を行っています．
Discord 上での活動が中心で，週末にオンラインで開催されるコンテストに参加しています．

具体的な活動内容は，[Activityリポジトリ](https://github.com/Kyoproers-at-USP/Activity) を参照してください．

また，[Kyoproers-at-USPリポジトリ](https://github.com/Kyoproers-at-USP/Kyoproers-at-USP) にて，広報関係の資料を公開しています．

プログラミング初心者でも大歓迎です！
